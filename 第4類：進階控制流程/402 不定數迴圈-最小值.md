# 402 不定數迴圈-最小值　不定数ループ-最小値
PYD402.py
## 設計說明：　プログラム説明：
請撰寫一程式，讓使用者輸入數字，輸入的動作直到輸入值為9999才結束，然後找出其最小值，並輸出最小值。

次の条件を満たすプログラムを作成せよ。
ユーザーに数字を入力させ、入力回数は9999と入力されるまで続き、
最終的に9999を除く全ての数値の中から最小値を割り出し、最小値を出力する事。


## 輸入輸出：　入出力：
輸入說明　入力説明
n個數值，直至9999結束輸入　数値をn個入力する、9999と入力されるまで続く

輸出說明　出力説明
n個數值中的最小值　n個の中の最小値

## 輸入輸出範例　入出力例
```
範例輸入　入力例
29
100
948
377
-28
0
-388
9999
範例輸出　出力例
-388
```
