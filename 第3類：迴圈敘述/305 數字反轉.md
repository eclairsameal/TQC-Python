# 305 數字反轉　数値の反転
PYD305.py
## 設計說明：　プログラム説明：
請撰寫一程式，讓使用者輸入一個正整數，將此數值以反轉的順序輸出。

次の条件を満たすプログラムを作成せよ。
ユーザーに自然数を1つ入力させ、その数値を反転し出力する事。

## 輸入輸出：　入出力：
輸入說明　入力説明
一個正整數　自然数1つ

輸出說明　出力説明
將此數值以反轉的順序輸出　入力した数値を反転させた数値

## 輸入輸出範例　入出力例
```
範例輸入1　入力例1
31283
範例輸出1　出力例1
38213
範例輸入2　入力例2
1003120
範例輸出2　出力例2
0213001
```
